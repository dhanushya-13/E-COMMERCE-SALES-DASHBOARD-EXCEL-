# Interactive E-Commerce Sales Dashboard
This project presents an interactive E-Commerce Sales Dashboard developed using Microsoft Excel.

## Objectives
- Analyze overall sales performance.
- Monitor key business metrics using KPI cards.
- Compare sales across countries and product categories.
- Identify top-performing products and brands.
- Build an interactive dashboard for business decision-making.

## Dashboard Features
- Interactive Excel Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Power Query Data Transformation
- Dynamic Slicers
- Sales Trend Analysis
- Country-wise Sales Analysis
- Product Category Analysis
- Brand-wise Performance Analysis

## Dataset

The dashboard is built using an E-Commerce Sales dataset containing transactional sales information.

### Dataset Attributes

- Order Date
- Country
- Product Category
- Product Sub-Category
- Brand
- Sales
- Profit
- Quantity
- Order ID

## Project Workflow

1. Import the dataset into Microsoft Excel.
2. Clean and transform the data using Power Query.
3. Create Pivot Tables for data summarization.
4. Build Pivot Charts for visualization.
5. Design KPI Cards using Excel formulas.
6. Add interactive slicers for filtering.
7. Develop the final interactive dashboard.
8. Generate business insights from the visualizations
